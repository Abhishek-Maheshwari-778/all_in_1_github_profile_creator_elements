## ⚡ **How to Build Your Profile (3 Steps)**

1.  **Repository Setup**: Create a new repository named `username/username`.
2.  **Choose a Base**: Go to [Templates](./Profile-Templates/) and pick one. Copy the raw markdown code.
3.  **Customize**:
    *   Paste the code into your `README.md`.
    *   Go to [Assets](./Assets_Collection/) and find GIFs/Icons you like.
    *   Replace the placeholder images with links to our assets or your own.
4.  **Commit**: Save your changes and watch your profile shine! ✨
